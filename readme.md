# Exploratory Data Analysis (EDA) on Titanic Dataset

## Objective
The objective of this experiment is to perform Exploratory Data Analysis (EDA) on the Titanic dataset to understand the data, identify patterns, detect missing values, and analyze the factors affecting passenger survival using statistical measures and visualization techniques.

## Tools and Technologies
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook / Google Colab

## Dataset
- Titanic Dataset (titanic.csv)

## Tasks Performed
- Loaded the Titanic dataset.
- Explored the dataset using Pandas.
- Checked dataset dimensions, column names, and data types.
- Identified missing values.
- Generated descriptive statistics.
- Performed correlation analysis.
- Visualized data using:
  - Count plots
  - Histograms
  - Box plots
  - Heatmap
  - Pair plot
- Analyzed relationships between passenger features and survival.
